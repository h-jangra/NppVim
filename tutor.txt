## Mode Switching
Esc - Normal mode
i - Insert mode
v - Visual mode
V - Visual line mode
Ctrl+q - Visual block mode
: - Command mode

## Movement
h j k l - Left Down Up Right
w W - Next word/WORD
b B - Previous word/WORD
e E - End of word/WORD
$ ^ 0 - End/Start/Start of line
{ } - Paragraph up/down
G - Go to line/end
:10 / 10G / 10gg - Go to line 10
gg - File start
H L - Top/Bottom of screen
Ctrl+d Ctrl+u - Page down/up

## Editing
x X - Delete char/backwards
dd - Delete line
D - Delete to line end
C - Change to line end
yy - Yank line
p P - Paste after/before
u - Undo
Ctrl+r- Redo
J - Join lines
r - Replace char
R - Replace mode
~ - Toggle case

## Operators
d - Delete (dw, d$, d%)
c - Change (cw, c$, c%)
y - Yank (yw, y$, y%)
> < - Indent/Unindent

## Text Objects
iw aw - Inner/A word
i" a" - Inside/Around quotes
i( a( - Inside/Around parens
i[ a[ - Inside/Around brackets
i{ a{ - Inside/Around braces
i< a< - Inside/Around angles
it at - Inside/Around tags
Test: { ( [  ] ) }

## Search
/ - Search forward
? - Regex search
n N - Next/Previous match
* # - Word under cursor forward/backward
f F - Find char forward/backward
t T - Till char forward/backward
; , - Repeat find reverse
% - Match bracket

## Marks
m[a-z] - Set local mark
m[A-Z] - Set global mark
mark - Jump to exact mark
''' - Jump back

## Visual Mode
d x - Delete selection
y - Yank selection
c - Change selection
> < - Indent/Unindent
~ - Toggle case
o - Swap cursor/anchor

## Command Mode
:w - Save
:q - Quit
:wq - Save & quit
:q! - Quit without save
:line - Go to line
:marks - Show marks
:delm marks or :dm marks- Delete marks
:delm! or dma or dm! - Delete all marks
:tutor or :tut- Open tutor

## Other
. - Repeat last change
gt gT - Next/Previous tab
zz - Center screen
gc - Toggle comment
[space or ]space - New line above/below

### Visual Mode
gq / gw – Wrap selected text
U / u – Convert selection to uppercase / lowercase
J – Join selected lines
r – Replace selected characters
S – Substitute selection with input text

### Normal Mode
( / ) – Move between sentences
M – Jump to middle of the screen
ge – Move to end of previous word
gE – Move to end of previous WORD
gf – Open file under cursor
gx – Open URL under cursor
J – Join lines without extra space
zt / zb – Scroll line to top / bottom of window
zc / zo – Close / open fold
gi – Jump to last insert position
[ / ] – Structured navigation (blocks / sections)
gU / gu – Uppercase / lowercase motion
gUU / guu – Uppercase / lowercase entire line
gd – Jump to first occurrence of symbol
Ctrl+E / Ctrl+Y – Scroll window down / up
Ctrl+F / Ctrl+B – Page forward / backward
gJ – Join lines without inserting space
S – Substitute entire line
| – Jump to specific column
gI – Insert at first non-blank character

## Command Mode
sort / sort! – Sort lines (normal / reverse)
sort n / sort n! – Numeric sort (normal / reverse)
noh / nohl / nohlsearch – Clear search highlights
set nu / set number – Enable line numbers
set nonu / set nonumber – Disable line numbers
set tw= – Set text width
wa / wall – Write all buffers
qa / qall – Quit all buffers
wqa / xa / xall – Save and quit all
e / edit – Edit file
bp / bprev / bprevious – Go to previous buffer
bd / bdelete – Delete buffer
sp / split – Horizontal split
vs / vsplit – Vertical split
tabn / tabp – Next / previous tab
tabnew – Open new tab
tabclose – Close current tab
